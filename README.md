## Overview

This repository contains a clean-room implementation of the [JMESPath Community](https://jmespath.site/) [specification](https://github.com/jmespath-community/jmespath.spec) in Rust.

## How-to build

```sh
cd src/
make build
make test
make coverage
make compliance
```
